usage: client.py [-h] [-u URL] [--token TOKEN] [-i {portaudio,ffmpeg,link}] [--print PRINT] [--list-available-languages] [--list-active-sessions] [--output-file OUTPUT_FILE] [-L] [-a AUDIODEVICE] [-ch AUDIOCHANNEL] [-f FFMPEG_INPUT] [--ffmpeg-pre FFMPEG_PRE]
                 [--ffmpeg-post FFMPEG_POST] [--volume VOLUME] [--ffmpeg-speed FFMPEG_SPEED] [--upload-video] [--translate_link] [--save-path SAVE_PATH] [--no-logging] [--run-mt RUN_MT] [--asr-kv ASR_PROPERTIES] [--no-textsegmenter] [--textseg-kv TEXTSEG_PROPERTIES]
                 [--use-error-correction] [--mt-kv MT_PROPERTIES] [--use-prep] [--prep-kv PREP_PROPERTIES] [--tts-kv TTS_PROPERTIES] [--video-kv VIDEO_PROPERTIES] [--show-on-website] [--website-title WEBSITE_TITLE] [--meta META] [--access ACCESS] [--run-scheduler]
                 [--timeout TIMEOUT] [--titanic-ip TITANIC_IP] [--run-tts RUN_TTS] [--generate-video GENERATE_VIDEO] [--save-video SAVE_VIDEO] [--summarize] [--absolute-timestamps] [--memory-words MEMORY_WORDS [MEMORY_WORDS ...]]

options:
  -h, --help            show this help message and exit
  -u URL, --url URL     Where to send the audio to
  --token TOKEN         Webapi access token for authentication
  -i {portaudio,ffmpeg,link}, --input {portaudio,ffmpeg,link}
                        Which input type should be used
  --print PRINT         specify amount of printing, 0: only hypos, 1: all recieved data, 2: all received data along with received timestamp (s), where start session timestamp = 0
  --list-available-languages
                        List available languages of mediator
  --list-active-sessions
                        List active session on mediator
  --output-file OUTPUT_FILE
                        Path to the file to save the output translations
  -L, --list            Pyaudio. List audio available audio devices
  -a AUDIODEVICE, --audiodevice AUDIODEVICE
                        Pyaudio. Index of audio device to use
  -ch AUDIOCHANNEL, --audiochannel AUDIOCHANNEL
                        index of audio channel to use (first channel = 1)
  -f FFMPEG_INPUT, --ffmpeg-input FFMPEG_INPUT
                        Input file/address that will be given to ffmpeg
  --ffmpeg-pre FFMPEG_PRE
                        ffmpeg options inserted before input parameter (-f).Don't forget to escape via string so this will be one single parameter.The parameter will be delimited at whitespace and does not support escaping
  --ffmpeg-post FFMPEG_POST
                        ffmpeg options inserted after input parameter (-f).Don't forget to escape via string so this will be one single parameter.The parameter will be delimited at whitespace and does not support escaping
  --volume VOLUME       Adjust the volume via ffmpeg
  --ffmpeg-speed FFMPEG_SPEED
                        set ffmpeg sending speed, -1 is infinite speed
  --upload-video        Wether to upload the full ffmpeg input video
  --translate_link      Wether to translate a link
  --save-path SAVE_PATH
                        Where to store the session in the archiv
  --no-logging          Do not log the session on the server
  --run-mt RUN_MT       Run a MT model in addition to ASR, comma separated string of output languages, e.g. "en-de,en-fr"
  --asr-kv ASR_PROPERTIES
                        Used asr properties, e.g. --asr-kv version=online --asr-kv segmenter=VAD --asr-kv stability_detection=False for online or --asr-kv version=offline --asr-kv segmenter=None for offline
  --no-textsegmenter    Set this to not use a textsegmenter
  --textseg-kv TEXTSEG_PROPERTIES
  --use-error-correction
  --mt-kv MT_PROPERTIES
                        Used mt properties, e.g. --mt-kv mode=SendStable --mt-kv mt_server=http://URL:PORT/SOMETHING
  --use-prep            Run a preprocessing model (e.g. noise filtering) before ASR
  --prep-kv PREP_PROPERTIES
                        Used prep properties
  --tts-kv TTS_PROPERTIES
                        Used tts properties
  --video-kv VIDEO_PROPERTIES
                        Used video properties
  --show-on-website     Wether to show this session on the website
  --website-title WEBSITE_TITLE
                        Which title is shown on the website
  --meta META           Meta information for website title
  --access ACCESS       Access information for website title
  --run-scheduler       Wether to run scheduler
  --timeout TIMEOUT     After how many seconds to stop the sending of audio, None: No timeout
  --titanic-ip TITANIC_IP
  --run-tts RUN_TTS     Run a TTS model, comma separated string of output languages, e.g. "en,de"
  --generate-video GENERATE_VIDEO
                        Run a video generation model, comma separated string of output languages, e.g. "en,de"
  --save-video SAVE_VIDEO
                        File to save the generated video locally on this pc to
  --summarize           Adds a summarizer after text segmentation.
  --absolute-timestamps
                        Returns absolute timestamps
  --memory-words MEMORY_WORDS [MEMORY_WORDS ...]
                        Words used in the memory-enhanced ASR model
